## O igri

Pilotiraj lebdeći brod kroz beskonačni hodnik ruševina starog sveta. Gravitacija te vuče nadole, hodnik se suzi, prepreke rastu. Svaki run traje 60–120 sekundi — ali uvek kažeš "još jedan".

## Kontrole

**Desktop:** `W` / `↑` — thrust gore · `A/D` / `←/→` — horizontalno · `Space` — start/restart

**Mobile:** Gornja polovina ekrana — thrust · Donje levo ◀ — levo · Donje desno ▶ — desno

## Kako se igra

1. Izbegavaj zidove i prepreke (brod ima kružni hitbox r=14px)
2. Skupljaj žute kristale — meta valuta koja se čuva između runova
3. Na svakom **checkpointu** (svakih ~400px) biraš jedan od dva power-upa
4. Posle smrti, troši kristale na trajne nadogradnje

## Nadogradnje

| Nadogradnja | Efekat | Max nivo |
|-------------|--------|----------|
| Brzina | +12% scroll po nivou | 3 |
| Štit | +1 auto-štit po runu | 2 |
| Magnet | +80px privlačenje kristala | 3 |
