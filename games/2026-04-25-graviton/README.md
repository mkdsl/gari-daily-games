# Graviton

**Žanr:** Platformer / Arkada (auto-scroll, gravity-flip)
**Datum:** 2026-04-25
**Play:** https://mkdsl.github.io/gari-daily-games/games/2026-04-25-graviton/

## Opis

Tvoj brod se sam kotrlja udesno. Jedino što možeš je **okrenuti gravitaciju** jednim pritiskom.
Prođi između šiljaka, blokova i rotirajućih oštrica — i nemoj da se zalepviš za pod ili plafon predugo.

## Kako se igra

- **Space / klik / tap** — obrće gravitaciju gore/dole
- Preživi što duže
- **G-overload**: ako si predugo na jednom polu (od zone 4), brod postaje crven i eksplodira — flipni pre nego što je kasno
- Svaki minut igra postaje brža
